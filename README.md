mongo_sync
==========

> Sync Remote and Local MongoDB Databases in Ruby!

## Installation

Add this line to your application's Gemfile:

    gem 'mongo_sync'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongo_sync
